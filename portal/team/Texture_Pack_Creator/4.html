Texture Pack Creator

Part 4

What folder and file should I add on a Texture Pack's folder?

Adding another file and folder on a Texture Pack's folder is giving what is Texture Pack's information. So we will dive in what file and folder you'll add inside the Texture Pack's folder.
This is your main Texture Pack's file contents, the first thing you add inside the Texture Pack's folder is:

1.

License Rules and Regulations and Lang file


Paste them inside your Texture Pack's folder. Download here.



2.

Make a description on Texts file


On en_GB.lang, make your own description what's about on the Texture Pack dramatically. We've a rules for making a description about on the Texture Pack...
The Cyan-Green line is the 1st description, the Red-Orange line is the 2nd description. Put the 2nd description on en_US.lang
The whole description will be on en_GB.lang



3.

Translate the whole description


Translate the whole description with supported languages given by every file inside. You can use Google Translate or Bing Translator depending on your division (learn how division works on RareFrostbites Team) and you can rearrange the grammar of translated language as long as you understand it, you don't need to fix it all, only what language do you speak and understand.

NOTE: If the description contains the title of the Texture Pack you use and/ or the name of RareFrostbites Team/ RareFrostbites, please do not translate it!
Supported Languages

  • Arabic = ar_SA
  • Bulgarian = bg_BG
  • Deutsch (German) = de_DE
  • English (US) = en_US + en_GB
  • Filipino = fil_PH
  • Hindi = hi_IN
  • Japan = ja_JP
  • Kannada = kn_IN
  • Korean (Hangul) = ko_KR
  • Portuguese = pt_PT
  • Tamil = ta_IN
  • Chinese (simplified) = zh_CN
